The '1-my_list' test module
=============================

---------------------------------------------------

First import the module:
>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()

---------------------
>>> my_list
[]

Test case (append ints):
------------------------
>>> my_list.append(1)
>>> my_list.append(8)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.print_sorted()
[1, 2, 4, 8]

[1, 8, 4, 2]


Test case (pop):
-----------------

2

[1, 4, 8]


Test case (insert):
--------------------
>>> my_list.insert(0, 5)

[5, 1, 8, 4]

----------------------------------
